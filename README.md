# kukso-nvim-dots
![Preview](screenshots/preview1.png)
![Smart search preview](screenshots/smartsearch_preview.png)

# Features:
- Smart search
- Cheat sheet
- Theme selector
- Simple and fast

# Plugins:
- Plugin manager: lazy.nvim
- Utilities: snacks.nvim, mini.nvim, mini.icons
- Lsp & completion: nvim-lspconfig blink.cmp, mason.nvim, mason-lspconfig.nvim, lazydev.nvim
- Interface: dracula.nvim, everforest, themery.nvim, transparent.nvim, bufferline.nvim, nvim-web-devicons
- Navigation & syntaxis: nvim-tree.lua, nvim-treesitter, nvim-navic

# Dependencies:
- System: neovim, git, make, unzip, gcc
- Fonts: JetBrainsMono Nerd Font

# Installation:
**Arch:**
`sudo pacman -S neovim git make unzip gcc ttf-jetbrains-mono-nerd`
`git clone https://github.com/Kuksoo/kukso-nvim-dots/ && mkdir ~/.config/nvim && mv kukso-nvim-dots ~/.config/nvim`
