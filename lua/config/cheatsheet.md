### Custom ###
- `jj`              : Quick ESC (insert -> normal)
- `<leader> sv`     : Reload current config (with notify)
- `<leader> ch`     : This window (Cheat Sheet)
- `Space `          : Leader key

### Files and Windows ###
- `<leader> e`      : File Explorer (NvimTree)
- `Ctrl + h/j/k/l`  : Move window focus
- `Tab / Shift+Tab` : Next / Previous buffer
- `<leader> x`      : Close current buffer (via Snacks)

### Search (Snacks Picker) ###
- `<space> <space>` : Smart Find (search everything)
- `<leader> ff`     : Find files by name
- `<leader> fg`     : Live Grep (search text in files)
- `<leader> fb`     : Active buffers (tabs)
- `<leader> ?`      : Search all active keymaps
- `<leader> un`     : Dismiss all notifications

### Vim Keybinds ###
- `i / a`           : Insert mode (before / after cursor)
- `o`               : New line below (insert mode)
- `w / b`           : Jump by words (Forward / Backward)
- `0 / $`           : Start / End of line
- `gg / G`          : Top / Bottom of file
- `u / Ctrl+r`      : Undo / Redo
- `/text`           : Search in file (`n` - next, `N` - prev)

### Commands ### 
- `:w / :q`         : Save / Quit
- `:Themery`        : Theme selector
- `:Lazy`           : Plugin manager
- `:Mason`          : LSP / Formatter manager
